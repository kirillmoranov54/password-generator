# Генератор паролей

Консольное приложение на Python для создания надёжных паролей.

**Возможности:**
- Режим быстрой генерации (18 символов со всеми типами символов)
- Полная настройка: количество, длина, цифры, буквы, спецсимволы
- Исключение неоднозначных символов (il1Lo0O)
- Автоматическое сохранение паролей на рабочий стол Windows
- Работает на ПК и на Android (через Termux)
